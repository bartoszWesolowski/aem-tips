data-sly-attribute

Adds attribute to host element

  • elements evaluated from left to right

    title="Title" data-sly-attribute.title="Title 1" data-sly-attribute.title="${'$'}{'Title evaluated'}" will evaluate to

    title="Title evaluated"

  • title="Title" data-sly-attribute.title="Title 1" data-sly-attribute.title="${'$'}{'Title evaluated'}" data-sly-attribute.title="${'$'}{null}"

    Will not render title attribute

  • data-sly-attribute.title="Title 1" data-sly-attribute.title="${'$'}{'Title evaluated'}" data-sly-attribute.title="${'$'}{null}" title="Title plain text"

    Will resolve to title="Title plain text"

  • class="${'$'}{''}" data-sly-attribute.id="${''}"

    Will not render any attribute

  • class="" data-sly-attribute.id=""

    Will not render only class="", id will not be rendered

  • This directive can also process a Map input to render multiple attributes

    Following map attrubutes will render

    • Key: '${key}' - value: '${model.attributes[key]}'

    attr1="attr1 value" attr2 attr-list="v1,v2"
