Replaces content of element with given text

  • All content of the element is replaced with given text. Given code: 

            
            <div data-sly-text="Text provided in data-sly-text">
              <p>some text in p element</p>
            </div>

    should render:

            
          <div>
            Text provided in data-sly-text
          </div>

    Actual code evaluated:

    some text in p element

  • If the text value is empty then content is replaced with empty text (nothing will be displayed)

            
           <div data-sly-text="${null}">
              <p>data-sly-text={null value}</p>
              <div>this will not be rendered</div>
            </div>

          ============ Output ============
            <div></div>

    Actual code evaluated:

    data-sly-text={null value}

    this will not be rendered

  • Default context of the element is set to text but it can be overriten

            
           <p data-sly-text="${'<strong>Bold and Proud</strong>' @ context='html'}"></p>

          ============ Output ============
           <p><strong>Bold and Proud</strong></p>

    Same example without context change

            
           <p data-sly-text="${'<strong>Bold and Proud</strong>'}"></p>
        
      
          ============ Output ============
      <strong>Bold and Proud</strong>
